## What Data This Tool Extracts / Produces  
| Field Name | Description |
|------------|-------------|
| ticker | Stock or security ticker being analyzed. |  
| date | Date of the data capture or financial snapshot. |  
| metrics | Object containing financial metrics such as P/E ratio, earnings growth, revenue, debt-to-equity, etc. |  
| forecast | AI-generated forecast for growth, valuation, or risk indicators. |  
| summary | AI-written summary of company financial health and outlook. |
